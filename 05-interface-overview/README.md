# 05. Обзор интерфейса

## Основные вкладки
- **Trade** — чарт + ордербук + панель открытия позиций.
- **Portfolio** — Positions, Open Orders, History, Unrealized PnL.
- **Markets** — список всех PERP.

## Элементы ордера
- Market / Limit / Post Only / Reduce Only.
- Слайдер leverage.
- Margin mode: Cross / Isolated (в продвинутом режиме).
