# 02. Настройка кошелька

## Рекомендуемый вариант
1. **RISE Wallet** (лучший опыт, zero gas).
2. MetaMask / Rabby / любой EVM-кошелёк.

## Как добавить сеть RISE Testnet
- Перейдите на [portal.risechain.com](https://portal.risechain.com).
- Нажмите **Add Network**.
- Добавьте вручную (Chain ID можно найти в документации).

## Подключение
1. Зайдите на testnet.rise.trade.
2. Нажмите **Connect Wallet**.
3. Выберите кошелёк и подтвердите.
4. Введите invite code (если требуется — берите в Discord).
