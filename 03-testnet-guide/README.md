# 03. Тестнет-гайд

## Как участвовать
1. Перейдите → [testnet.rise.trade](https://testnet.rise.trade).
2. Подключите кошелёк.
3. Нажмите **Faucet** → Claim USDC (часто $1000 ежедневно для соревнований).
4. Перейдите в раздел **Competition** (Trading Titans) — подключите X-аккаунт.

## Что торговать
- BTC-PERP, ETH-PERP, SOL-PERP и другие.
- Плечо до 50x.

**Совет**: Торгуйте каждый день — это помогает в наборе поинтов для возможного airdrop.
