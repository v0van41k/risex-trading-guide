# 07. Продвинутая торговля

## Portfolio Margin
- Вся позиция и collateral считаются вместе → более эффективное использование капитала.

## AutoYield
- Idle collateral автоматически зарабатывает yield.

## Order types
- **Post Only** — только maker.
- **Reduce Only** — только уменьшение позиции.
- Conditional orders (в разработке).

## Modular Sub-accounts
- Создавайте отдельные программируемые аккаунты под разные стратегии.
