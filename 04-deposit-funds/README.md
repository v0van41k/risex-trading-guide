# 04. Депозит средств

## На Testnet
- Используйте faucet (USDC, ETH и другие токены).
- Ежедневный рефилл в соревнованиях.

## На Mainnet (после запуска)
- Bridge с Ethereum / других L2.
- Поддерживаются любые ERC20 как collateral.
- AutoYield автоматически зарабатывает на idle средствах.

**Важно**: Всегда проверяйте официальные bridge-ссылки.
