# 06. Базовая торговля

## Как открыть позицию
1. Выберите пару (например BTC-PERP).
2. Выберите **Long** или **Short**.
3. Установите leverage (10-50x).
4. Введите сумму (USDC).
5. Нажмите **Open Position**.

## Закрытие
- Кнопка **Close** или **%** (25/50/75/100%).
- Используйте Limit order для лучшей цены.

**Пример**: 1000 USDC с 20x → контроль 20 000 USDC позиции.
